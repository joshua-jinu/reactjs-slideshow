- In this exercise, we will also explore material ui for decoration purpose.

Starter Code:

NOTE This project was created using npm create vite@latest command - which means all the necessary modules was installed when this command got executed. So, if you  start making changes in the required files, you will not able to see any changes, or you may ran into errors like - module not found. The main reason for this is that -- inside your project you are trying to use some modules, which your system may not have, so you need to install those modules first.
